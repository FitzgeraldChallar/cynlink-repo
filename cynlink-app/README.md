## Getting Started

Follow these steps to clone and run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cynlink-app.git
cd cynlink-app
```

### 2. Install dependencies

```bash
npm install
# or
yarn
# or
pnpm install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application running.

### 4. Start editing

The main entry point is `app/page.tsx`. Changes you make here or in any components will automatically refresh in the browser.
